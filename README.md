
# Shape judger

This is an AI that will judge the shape is rectangle or ellipse.

Most made for my personal exppppp.

Requires cupy.

## How to run

If u use windows or mac, run this cmd in ur terminal.

```bash
$ python Learn.py
```


For linux,

```bash
$ python3 Learn.py
```

## How to clean the data

If u use windows or mac, run this cmd in ur terminal.

```bash
$ python ResetData.py
```


For linux,

```bash
$ python3 ResetData.py
```

## How to see the plot

If u use windows or mac, run this cmd in ur terminal.

```bash
$ python ShowPlot.py
```


For linux,

```bash
$ python3 ShowPlot.py
```
